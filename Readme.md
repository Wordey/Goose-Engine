# The Goose Engine
welcome, to the goose engine project it's mini open-source game engine

# Compiling
i recomend for you use any linux distro(i use artix btw) and gcc(if you like you can rewrite makefile for clang)
```bash
    git clone https://www.github.com/Wordey/Goose-Engine.git
    cd Goose-Engine
    make
```
